Weibull Probability Plot

This procedure produces a Weibull plot for a set of failure and suspension data. Each case includes a time value, a type value (failure or suspension), and a count of cases of that time and count. A suspension represents cases that have not yet failed, i.e., the data are right censored.

Time Select the variable that indicates the time for this case. Times must be positive. They may indicate standard calendar quantities such as months, or they may be in terms of engineering quantities such as usage cycles. This is not an SPSS time variable.

Counts Optionally select a variable holding the count of instances for this case. If omitted, a count of 1 is assumed.

Failures and Suspensions Optionally specify a variable to indicate the type of the case: failures or suspensions. If a variable is not selected, all cases are assumed to be failures.

Failure Code and Suspension Code Indicate the string that indicates whether the case is a failure or suspension.

• Cases with any other values are discarded from the analysis.

ID Variable Optionally specify a string variable whose value (first case) should be used to label the output.

Options

Chart Title Specify a title for the chart. If the text )ID occurs in the title text and an id variable is specified, the value of that variable will be inserted into the chart title at that point.

Annotation Specify an annotation to appear on the X axis along with the time label. The current date and time are automatically appended to the annotation.

Save

Graph Data Dataset Name Specify a dataset name in order to produce a dataset containing the data plotted in the graph.

Graph Data Sav File Specify a file name, including the sav extension, in order to produce a data file containing the data plotted in the graph. The contents will be the same as the dataset.

Graph Data Text File Specify a file name in order to produce a file in csv format containing the data plotted in the graph.

Text File Mode If creating a text file, specify whether to overwrite previous contents or append to the file. If appending, the structure, including the presence or absence of an id variable, must be the same as the previous contents.

Additional Features

This dialog generates syntax for the STATS WEIBULL PLOT extension command. To display help for this command, run the following syntax:

STATS WEIBULL PLOT /HELP.

Requirements

This procedure requires SPSS Statistics version 20 with FixPack1 or later. It can also be run with a graphics hot fix for Version 19. Contact Technical Support for the hotfix.

The procedure requires the Python Essentials and the STATS WEIBULL PLOT extension command. Note that installing the extension requires an extra step to set up the chart template file. See the instructions in the extension readme file for details.
