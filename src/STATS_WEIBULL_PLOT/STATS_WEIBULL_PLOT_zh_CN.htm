威布尔概率图

此过程生成一组故障和挂起数据的威布尔图。每个个案都包含一个时间值、一个类型值（故障或挂起）以及该时间和类型的个案计数。 挂起表示尚未失败的个案，即，数据被右检剔。

时间 请选择指示此个案的时间的变量。时间必须是正数。时间可能指示月份数之类的标准日历数量，也可能用工程量（例如使用周期）表示。这不是 SPSS 时间变量。

计数 （可选）请选择用于存放此个案的实例计数的变量。如果省略此项，那么将假定计数为 1。

故障和挂起 （可选）请指定用于指示个案类型为故障还是挂起的变量。如果未选择变量，那么所有个案都假定为故障。

故障代码和挂起代码

请指定表示个案是故障还是挂起的字符串。

* 分析中将废弃具有任何其他值的个案。

ID 变量 （可选）请指定其值（第一个个案）应该用来对输出进行标注的字符串变量。

选项

图表标题 请指定图表的标题。如果文本 )ID 出现在标题文本中，并且指定了 ID 变量，那么该变量的值将插入到图表标题中的该位置处。

注释 请指定将与时间标签一起显示在 X 轴上的注释。当前日期和时间将自动追加到该注释。

保存(V)

图形数据数据集名称 请指定数据集名称，以生成包含图形中绘制的数据的数据集。

图形数据 Sav 文件 请指定文件名（包括 sav 扩展名），以生成包含图形中绘制的数据的数据文件。 内容将与数据集相同。

图形数据文本文件 请指定文件名，以生成包含图形中绘制的数据的 csv 格式的文件。

文本文件方式 如果创建文本文件，请指定是覆盖先前的内容还是附加到文件。 如果附加，那么结构（包括存在或不存在标识变量）必须与先前内容相同。

其他功能部件

此对话框将生成 STATS WEIBULL PLOT 扩展命令的语法。要显示此命令的帮助，请运行以下语法：

STATS WEIBULL PLOT /HELP.

要求

此过程需要带有 FP1 的 SPSS Statistics V20 或更高版本。此过程还可以在安装了 V19 的图形最新修订程序的情况下运行。请与技术支持人员联系以获取最新修订程序。

此过程需要 Python Essentials 和 STATS WEIBULL PLOT 扩展命令。请注意，安装扩展需要额外的步骤来设置图表模板文件。有关详细信息，请参阅扩展自述文件中的指示信息。
