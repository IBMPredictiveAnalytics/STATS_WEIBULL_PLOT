Gráfico de probabilidade de Weibull

Este procedimento produz um gráfico de Weibull para um conjunto de dados de falha e de suspensão. Cada caso inclui um valor de tempo, um valor de tipo (falha ou suspensão) e uma contagem de casos desse tempo e contagem. A suspensão representa os casos que ainda não falharam, ou seja, os dados estão censurados à direita.

Tempo Selecione a variável que indica o tempo para este caso. Tempos devem ser positivos. Eles podem indicar as quantidades do calendário padrão, como meses, ou podem ser em termos de quantidades de engenharia, como ciclos de uso. Não é uma variável de tempo do SPSS.

Contagens Opcionalmente, selecione uma variável que retém a contagem de instâncias para este caso. Se omitida, uma contagem igual a 1 será assumida.

Falhas e suspensões Opcionalmente, especifique uma variável para indicar o tipo do caso: falhas ou suspensões. Se uma variável não for selecionada, todos os casos serão assumidos como falhas.

Código de falha e código de suspensão

Indicam a sequência de caracteres que indica se o caso é uma falha ou uma suspensão.

* Os casos com quaisquer outros valores são descartados da análise.

Variável de ID Opcionalmente, especifique uma variável de sequência de caracteres cujo valor (primeiro caso) deve ser usado para rotular a saída.

Opções

Título do gráfico Especifique um título para o gráfico. Se o texto )ID ocorrer no texto do título e uma variável de ID for especificada, o valor dessa variável será inserido no título do gráfico neste ponto.

Anotação Especifique uma anotação para aparecer no eixo X juntamente com o rótulo de tempo. A data e a hora atuais são automaticamente anexadas à anotação.

Salvar

Nome do conjunto de dados do gráfico Especifique um nome do conjunto de dados para produzir um conjunto de dados que contenha os dados usados para criar o gráfico.

Arquivo Sav de dados gráficos: especifica um nome de arquivo, incluindo a extensão sav, para produzir um arquivo de dados contendo os dados plotados no gráfico. O conteúdo será o mesmo do conjunto de dados.

Arquivo de texto de dados gráficos: especifica um nome de arquivo para produzir um Arquivo no formato csv contendo os dados plotados no gráfico.

Modo de arquivo de texto: se estiver criando um arquivo de texto, especifique se irá sobrescrever o conteúdo anterior ou anexar ao arquivo. Se anexar, a estrutura, incluindo a presença ou ausência de uma variável id, deverá ser igual ao conteúdo anterior.

Recursos adicionais

Este diálogo gera sintaxe para o comando de extensão STATS WEIBULL PLOT. Para exibir ajuda para este comando, execute a seguinte sintaxe:

STATS WEIBULL PLOT /HELP.

Requisitos

Este procedimento requer o SPSS Statistics versão 20 com FixPack1 ou posterior. Ele também pode ser executado com um hotfix de gráfico para a Versão 19. Entre em contato com o Suporte técnico para obter o hotfix.

O procedimento requer o Python Essentials e o comando de extensão STATS WEIBULL PLOT. Observe que a instalação da extensão requer uma etapa extra para configurar o arquivo de modelo de diagrama. Consulte as instruções no arquivo leia-me de extensão, para obter detalhes.
