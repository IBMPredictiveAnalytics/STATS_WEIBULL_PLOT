Tracé de probabilités de Weibull

Cette procédure génère un tracé de Weibull pour un jeu de données de défaillances et d'interruptions. Chaque observation inclut une valeur d'heure, une valeur de type (échec ou interruption) et un nombre d'observations de cette heure et de ce nombre. Les interruptions correspondent aux observations qui n'ont pas encore échoué, c'est-à-dire, les observations dont les données de droite sont censurées.

Heure : Sélectionnez la variable qui indique l'heure de l'observation. Les heures doivent être positives. Elles peuvent indiquer des quantités standard, telles que des mois, ou des quantités d'ingénierie, telles que des cycles d'utilisation. Il ne s'agit pas d'une variable d'heure SPSS.

Effectifs : Si vous le souhaitez, sélectionnez la variable qui détient le nombre d'instances pour cette observation. Dans le cas contraire, un effectif de 1 est pris en compte.

Défaillances et interruptions : Si vous le souhaitez, spécifiez une variable pour indiquer le type de l'observation (défaillance ou interruption). Si aucune variable n'est sélectionnée, toutes les observations sont considérées comme étant défaillantes.

Code de défaillance et code d'interruption :

Indiquez la chaîne qui détermine si l'observation est défaillante ou interrompue.

* Les observations comportant d'autres valeurs sont supprimées de l'analyse.

Variable ID : Si vous le souhaitez, vous pouvez spécifier une variable de chaîne dont la valeur (première observation) doit être utilisée pour étiqueter la sortie.

Options

Titre du graphique : Attribuez un titre au graphique. Si le texte )ID apparaît dans le texte du titre et qu'une variable ID est spécifiée, la valeur de cette variable est insérée dans le titre du graphique à ce point.

Annotation : Indiquez l'annotation à afficher sur l'axe des X avec le libellé Heure. La date et l'heure en cours sont automatiquement ajoutés à l'annotation.

Sauvegarder

Nom du jeu de données graphiques : Indiquez le nom du jeu de données afin de générer le jeu de données contenant les données tracées dans le graphique.

Fichier sav des données graphiques : Spécifiez un nom de fichier, avec l'extension sav, pour générer un fichier de données contenant les données tracées dans le graphique. Le contenu est identique au jeu de données.

Fichier texte des données graphiques : Spécifiez un nom de fichier pour générer un fichier au format csv contenant les données tracées dans le graphique.

Mode du fichier texte : Si vous créez un fichier texte, indiquez si le contenu précédent doit être remplacé ou si le nouveau contenu doit être ajouté au fichier. Dans ce second cas, la structure, avec la présence ou l'absence d'une variable id, doit être identique à celle du contenu précédent.

Fonctions supplémentaires

Cette boîte de dialogue génère une syntaxe pour la commande d'extension STATS WEIBULL PLOT. Pour afficher l'aide de cette commande, exécutez la syntaxe suivante :

STATS WEIBULL PLOT /HELP.

Conditions requises

Cette procédure requiert SPSS Statistics version 20 avec le groupe de correctifs 1 ou ultérieur. Elle peut également être exécutée avec un correctif logiciel de graphiques pour la version 19. Contactez le support technique pour obtenir le correctif logiciel.

La procédure requiert Python Essentials et la commande d'extension STATS WEIBULL PLOT. Notez que l'installation de l'extension requiert une étape supplémentaire pour configurer le fichier modèle de graphique. Pour plus de détails, reportez-vous aux instructions du fichier Readme de l'extension.
