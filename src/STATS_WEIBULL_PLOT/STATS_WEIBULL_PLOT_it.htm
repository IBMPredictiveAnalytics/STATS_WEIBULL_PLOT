Grafico di probabilità di Weibull

Questa procedura produce un grafico di Weibull per un insieme di dati di errore e di sospensione. Ogni caso include un valore temporale, un valore tipo (errore o sospensione) e il numero dei casi di quel periodo e conteggio. Una sospensione rappresenta i casi che non hanno ancora esito negativo, ossia, i dati vengono censurati a destra.

Ora Selezionare la variabile che indica l'ora per questo caso. I valori temporali devono essere positivi. Possono indicare quantità del calendario standard, come i mesi, o essere termini di quantità di engineering, come i cicli di utilizzo. Non è una variabile tempo di SPSS.

Conteggi Facoltativamente, selezionare una variabile che contiene il conteggio delle istanze per questo caso. Se omesso, viene presunto il conteggio 1.

Errori e sospensioni Facoltativamente, è possibile specificare una variabile per indicare il tipo del caso: errori o sospensioni. Se non è selezionata una variabile, si presume che tutti i casi sono errori.

Codice errore e codice sospensione

Indicare la stringa che indica se il caso è un errore o una sospensione.

* I casi con qualsiasi altro valore sono eliminati dall'analisi.

Variabile ID Facoltativamente, specificare una variabile stringa il cui valore (primo caso) deve essere utilizzato per etichettare l'output.

Opzioni

Titolo del grafico Specificare un titolo per il grafico. Se compare il testo )ID nel titolo ed è specificata una variabile ID, il valore della variabile verrà inserito nel titolo del grafico in quel punto.

Annotazione Specificare un'annotazione da visualizzare sull'asse X insieme all'etichetta ora. La data e l'ora correnti vengono accodate automaticamente all'annotazione.

Salva

Nome del dataset di dati grafici Specificare un nome dataset per poter creare un dataset contenente i dati tracciati nel grafico.

File sav dati grafici Specificare un nome file, includendo l'estensione sav, per produrre un file di dati che contiene i dati tracciati nel grafico. Il contenuto sarà lo stesso del dataset.

File di testo dati grafici Specificare un nome file per produrre un file in formato csv che contiene i dati tracciati nel grafico.

Modalità file di testo Se si sta creando un file di testo, specificare se sovrascrivere il contenuto precedente o se accodare al file. In caso di accodamento, la struttura, compresa la presenta o l'assenza di una variabile ID, deve essere la stessa del contenuto precedente.

Funzioni aggiuntive

Questa finestra di dialogo genera la sintassi per il comando di estensione STATS WEIBULL PLOT. Per visualizzare la guida per questo comando, utilizzare la sintassi riportata di seguito:

STATS WEIBULL PLOT /HELP.

Requisiti

Questa procedura richiede SPSS Statistics versione 20 con FixPack1 o successivo. È anche possibile utilizzare una hot fix grafica per la Versione 19. Rivolgersi al supporto tecnico per la hotfix.

La procedura richiede Python Essentials e il comando di estensione STATS WEIBULL PLOT. Nota: l'installazione dell'estensione richiede una fase supplementare per configurare il file modello di grafico. Per i dettagli, consultare le istruzioni nel file readme dell'estensione.
