Weibull 機率圖

此程序會產生失敗及暫停資料集的 Weibull 圖。每一個觀察值都包含時間值、類型值（失敗或暫停），以及該時間及計數的觀察值計數。暫停代表尚未失敗的觀察值，即資料為向右設限。

時間 選取指出此觀察值時間的變數。 時間必須是正數。它們可能指出標準行事曆數量（如月數），或可能為工程數量術語（如使用週期）。這不是 SPAA 時間變數。

計數 選擇性地選取保留此觀察值實例計數的變數。如果已忽略，則假設計數為 1。

失敗及暫停 選擇性地指定變數，以指出觀察值的類型：失敗或暫停。 如果未選取變數，則所有觀察值都假設為失敗。

失敗碼及暫停碼

指出字串，其可指出觀察值為失敗還是暫停。

* 透過分析會捨棄具有任何其他值的觀察值。

ID 變數 選擇性地指定字串變數，其值（第一個觀察值）應用於標記輸出。

選項

圖表標題 指定圖表的標題。如果文字 )ID 發生在標題文字中，且已指定 ID 變數，則該變數的值會插入圖表標題的該點上。

註釋 指定註釋，以隨時間標籤，在 X 軸上顯示。現行日期和時間會自動附加至註釋。

儲存(V)

圖形資料的資料集名稱 指定資料集名稱，以便產生包含圖形中繪製之資料的資料集。

圖形資料 SAV 檔 指定檔名（其中包括 SAV 副檔名），從而產生資料檔，其中包含在圖形中繪製的資料。內容將與資料集相同。

圖形資料文字檔 指定檔名，從而產生 CSV 格式的檔案，其中包含在圖形中繪製的資料。

文字檔模式 如果建立文字檔，則指定是改寫先前的內容，還是附加到檔案。如果附加，則結構（包括存在或不存在 ID 變數）必須與先前的內容相同。

其他特性

此對話框會產生適用於 STATS WEIBULL PLOT 延伸指令的語法。若要顯示此指令的說明，請執行下列語法：

STATS WEIBULL PLOT /HELP.

需求

此程序需要 SPSS Statistics 第 20 版本（含 Fix Pack 1）或更新版本。它還可以與第 19 版的圖形緊急修復程式搭配執行。如需該緊急修復程式，請與技術支援中心聯絡。

此程序需要 Python Essentials 及 STATS WEIBULL PLOT 延伸指令。請注意，安裝延伸需要額外的步驟，以設定圖表範本檔案。請參閱延伸 Readme 檔中的指示，以取得詳細資料。
