ワイブル確率プロット

この手順では、1 セットの故障データと停止データに関するワイブル プロットを作成します。各ケースには、時間値、タイプ値 (故障または停止)、およびその時間と度数のケースの度数が含まれています。停止は、まだ故障していないケース、つまりデータが右側打ち切りであることを表します。

時間: このケースの時間を示す変数を選択します。 時間は正の数値でなければなりません。時間は標準のカレンダー数量 (月など) で示される場合や、技術的な数量 (使用サイクルなど) で示される場合があります。これは SPSS の時間変数ではありません。

度数: 必要に応じて、このケースのインスタンスの度数を保持している変数を選択します。省略すると、度数 1 が想定されます。

故障と停止: 必要に応じて、ケースのタイプ (故障または停止) を示す変数を指定します。 変数を選択しなかった場合は、すべてのケースが故障と想定されます。

故障コードと停止コード:

ケースが故障か停止かを示す文字列を示します。

* これ以外の値を持つケースは分析から除外されます。

ID 変数: 必要に応じて、出力にラベルを付けるために使用される値 (最初のケース) を持つ文字型変数を指定します。

オプション

図表の表題: 図表の表題を指定します。テキスト「)ID」が表題に含まれている場合に ID 変数を指定すると、その変数の値が図表の表題のその位置に挿入されます。

注釈: X 軸に時間ラベルとともに表示される注釈を指定します。注釈には、現在の日付と時刻が自動的に付加されます。

保存

グラフ データのデータセット名: グラフでプロットされるデータを含んでいるデータセットを作成する場合に、データセット名を指定します。

グラフ データの sav ファイル グラフにプロットしたデータを含むデータ ファイルを作成するためのファイル名を、sav の拡張子も含めて指定します。内容はデータセットと同じになります。

グラフ データのテキスト ファイル グラフにプロットしたデータを含む csv 形式ファイルを生成するためのファイル名を指定します。

テキスト ファイル モード テキスト ファイルを作成する場合に、以前の内容を上書きするか、ファイルに追加するかを指定します。 追加する場合は、ID 変数の有無も含め、以前の内容と同じ構造でなければなりません。

その他の機能

このダイアログは、STATS WEIBULL PLOT 拡張コマンドのシンタックスを生成します。このコマンドのヘルプを表示するには、次のシンタックスを実行します。

STATS WEIBULL PLOT /HELP.

要件

この手順には、SPSS Statistics バージョン 20 フィックスパック 1 以降が必要です。また、グラフィックのホット フィックスが適用されたバージョン 19 でも実行できます。ホット フィックスについては、テクニカル サポートにお問い合わせください。

この手順には、Python Essentials と STATS WEIBULL PLOT 拡張コマンドが必要です。 この拡張機能をインストールするには、図表テンプレート ファイルを別途セットアップする必要があります。詳しくは、拡張機能の readme ファイルに記載されている説明を参照してください。
