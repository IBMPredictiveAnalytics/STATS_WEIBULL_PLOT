Диаграмма вероятности Вейбулла

Эта процедура создает диаграмму Вейбулла для набора данных об ошибках и приостановках. Каждое наблюдение включает значение времени, значение типа (ошибка или приостановка) и количество наблюдений с таким временем и типом. Приостановка представляет собой наблюдения, в которых ошибка еще не произошла, то есть данные правильно отцензурированы.

Время Выберите переменную, обозначающую время для этого наблюдения. Значения времени должны быть положительными. Они могут соответствовать стандартным календарным отсчетам, таким как месяцы, или строиться по функциональным показателям, например, по циклам использования. Это не переменная времени SPSS.

Отсчеты Дополнительно можно выбрать переменную, содержащую количество экземпляров для данного наблюдения. Если эта переменная отсутствует, по умолчанию предполагается значение 1.

Ошибки и приостановки Дополнительно задайте переменную, обозначающую тип наблюдения: ошибки или приостановки. Если эта переменная не выбрана, все наблюдения предполагаются ошибками.

Код ошибки и код приостановки

Строка, обозначающая тип наблюдения, ошибка это или приостановка.

* Наблюдения с любыми другими значениями не рассматриваются при анализе.

Переменная ID Необязательно, укажите строковую переменную, значение которой (первое наблюдение) должно использоваться для метки вывода.

Параметры

Заголовок диаграммы Задайте заголовок для диаграммы. Если в тексте заголовка есть символы )ID и задана переменная ID, значение этой переменной будет вставлено в заголовок диаграммы в этой точке.

Аннотация Задайте аннотацию, которая появится на оси X вместе с меткой времени. К аннотации автоматически присоединяется текущая дата и время.

Сохранить

Имя набора данных диаграммы Задайте имя для создания набора данных, который содержит данные, представляемые на диаграмме.

Файл Sav графических данных Укажите имя файла, включая расширение sav, чтобы произвести файл данных, содержащий данные, показанные на графике. Содержимое будет совпадать с набором данных.

Текстовый файл графических данных Укажите имя файла, чтобы произвести файл в формате CSV, содержащий данные, показанные на графике.

Режим текстовых файлов При создании текстового файла укажите, нужно ли перезаписать предыдущее содержимое или добавить в файл. В случае добавления структура, включая наличие или отсутствие переменной ID, должна быть такой же, как и предыдущее содержимое.

Дополнительные возможности

Это диалоговое окно генерирует синтаксис для команды расширения STATS WEIBULL PLOT. Чтобы вывести справку для этой команды, введите следующую команду:

STATS WEIBULL PLOT /HELP.

Технические требования

Для этой процедуры требуется SPSS Statistics версии 20 с Fix Pack 1 или новее. Она может выполняться также в Версии 19, если установлен оперативный пакет исправлений для графики. Обратитесь к группе технической поддержки за этим пакетом.

Для этой процедуры требуется пакет Python Essentials и команда расширения STATS WEIBULL PLOT. Обратите внимание, что установка расширения требует дополнительного шага для настройки файла шаблона диаграммы. Подробности смотрите в инструкциях в файле readme расширения.
