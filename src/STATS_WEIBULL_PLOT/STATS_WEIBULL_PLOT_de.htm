Weibull-Wahrscheinlichkeitsdiagramm

Diese Prozedur erstellt ein Weibull-Diagramm für ein Set von Fehler- und Suspensionsdaten. Jeder Fall umfasst einen Zeitwert, einen Typwert (Fehler oder Suspension) und die Anzahl der Fälle mit dieser Zeit und dieser Anzahl. Bei einer Suspension handelt es sich um Fälle, die noch nicht fehlgeschlagen sind, aber deren Daten rechtszensiert wurden.

Zeit Wählen Sie die Variable aus, die die Zeit für diesen Fall angibt. Die Zeiten müssen positiv sein. Sie können in den üblichen Kalendereinheiten wie Monate oder in technischen Mengeneinheiten wie Nutzungszyklen ausgedrückt werden. Dabei handelt es sich nicht um eine SPSS-Zeitvariable.

Anzahl Wählen Sie optional eine Variable aus, die die Anzahl der Instanzen für diesen Fall enthält. Wenn keine Angabe gemacht wird, wird eine Anzahl von 1 angenommen.

Fehler und Suspensionen Geben Sie optional eine Variable an, um den Typ des Falls (Fehler oder Suspensionen) anzugeben. Wenn keine Variable ausgewählt wird, werden alle Fälle als Fehler eingestuft.

Fehlercode und Suspensionscode

Geben Sie die Zeichenfolge an, die darauf hinweist, ob der Fall als Fehler oder Suspension zu werten ist.

* Fälle mit anderen Werten werden aus der Analyse gelöscht.

ID-Variable Geben Sie optional eine Zeichenfolgevariable an, deren Wert (erster Fall) zur Beschriftung der Ausgabe verwendet werden soll.

Optionen

Diagrammtitel Geben Sie einen Titel für das Diagramm an. Wenn der Text )ID im Titeltext vorkommt und eine ID-Variable angegeben ist, wird an dieser Stelle im Diagrammtitel der Wert dieser Variable eingefügt.

Anmerkung Geben Sie eine Anmerkung an, die zusammen mit der Zeitbeschriftung auf der X-Achse dargestellt werden soll. An die Anmerkung werden automatisch die aktuelle Datums- und Zeitangabe angehängt.

Speichern

Name des Datasets für Diagrammdaten Geben Sie einen Datasetnamen an, damit ein Dataset erstellt wird, das die im Diagramm dargestellten Daten enthält.

sav-Datei für Diagrammdaten Geben Sie einen Dateinamen mit der Erweiterung sav an, damit eine Datendatei erstellt wird, die die im Diagramm dargestellten Daten enthält. Der Inhalt ist derselbe wie beim Dataset.

Textdatei für Diagrammdaten Geben Sie einen Dateinamen an, um eine Datei im CSV-Format zu erstellen, die die im Diagramm dargestellten Daten enthält.

Textdateimodus Wenn Sie eine Textdatei erstellen, geben Sie an, ob der vorherige Inhalt überschrieben oder an die Datei angehängt werden soll. Beim Anhängen muss die Struktur, einschließlich des Vorhandenseins oder Nichtvorhandenseins einer id-Variablen, mit dem vorherigen Inhalt übereinstimmen.

Weitere Funktionen

Über dieses Dialogfeld wird Syntax für den Erweiterungsbefehl STATS WEIBULL PLOT generiert. Führen Sie folgende Syntax aus, um Hilfe für diesen Befehl anzuzeigen:

STATS WEIBULL PLOT /HELP.

Anforderungen

Für diese Prozedur ist SPSS Statistics Version 20 mit Fixpack 1 oder höher erforderlich. Sie kann auch mit einem Grafikhotfix für Version 19 ausgeführt werden. Der Hotfix kann über den Technical Support bezogen werden.

Für die Prozedur sind Python Essentials und der Erweiterungsbefehl STATS WEIBULL PLOT erforderlich. Beachten Sie, dass die Installation der Erweiterung einen zusätzlichen Schritt erfordert, um die Diagrammvorlagendatei einzurichten. Einzelheiten finden Sie in den Anweisungen in der Readme-Datei der Erweiterung.
