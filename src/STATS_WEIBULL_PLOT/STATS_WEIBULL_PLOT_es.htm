Gráfico de probabilidad de Weibull

Este procedimiento genera un gráfico de Weibull de una serie de datos de errores y suspensión. Cada caso incluye un valor de tiempo, un valor de tipo (fallo o suspensión) y un recuento de casos de dicho tiempo y recuento. Una suspensión representa los casos que aún no han fallado, es decir, en que los datos están censurados por la derecha.

Tiempo Seleccione la variable que indica el tiempo de este caso. Los tiempos deben ser positivos. Pueden indicar cantidades de calendario estándar tales como meses, o pueden ser en términos de cantidades de ingeniería tales como ciclos de uso. No es lo mismo que la variable de tiempo de SPSS.

Recuentos Opcionalmente, puede seleccionar una variable que retenga el recuento de instancias de este caso. Si se omite, se presupone un recuento igual a 1.

Errores y suspensiones Opcionalmente, puede especificar una variable para indicar el tipo del caso: errores o suspensiones. Si no se selecciona una variable, se supone que todos los casos son errores.

Código de error y código de suspensión

Señala la cadena que indica si el caso es un error o una suspensión.

* Los casos con otros valores se descartan del análisis.

Variable de ID Opcionalmente, especifique una variable de cadena cuyo valor (primer caso) se tenga que utilizar para etiquetar el resultado.

Opciones

Título de gráfico Especifique un título para el gráfico. Si aparece el texto )ID dentro del texto de título y se ha especificado una variable de ID, se insertará el valor de esa variable dentro del título del gráfico en ese punto.

Anotación Especifique la anotación que aparecerá en el eje X junto con la etiqueta de hora. La fecha y hora actuales se añaden automáticamente a la anotación.

Guardar

Nombre de conjunto de los datos del gráfico Especifique un nombre de conjunto de datos para generar un conjunto de datos que contenga los datos representados en el gráfico.

Archivo Sav de datos de gráfico Especifique un nombre de archivo, incluida la extensión sav, para producir un archivo de datos que contenga los datos representados en el gráfico. El contenido será el mismo que el conjunto de datos.

Archivo de texto de datos del gráfico Especifique un nombre de archivo para producir un archivo en formato csv que contenga los datos representados en el gráfico.

Modo de archivo de texto Si crea un archivo de texto, especifique si se debe sobrescribir el contenido anterior o si se debe añadir al archivo. Si se añade, la estructura, incluida la presencia o ausencia de una variable de ID, debe ser la misma que la del contenido anterior.

Características adicionales

Este diálogo genera sintaxis para el comando de extensión STATS WEIBULL PLOT. Para mostrar la ayuda de este comando, ejecute la siguiente sintaxis:

STATS WEIBULL PLOT /HELP.

Requisitos

Este procedimiento requiere SPSS Statistics versión 20 con el FixPack1 o posterior. También se puede ejecutar con un hotfix para gráficos para la versión 19. Póngase en contacto con el soporte técnico para obtener el hotfix.

El procedimiento requiere Python Essentials y el comando de extensión STATS WEIBULL PLOT. Tenga en cuenta que para instalar la extensión se requiere un paso adicional para configurar el archivo de plantilla de gráfico. Consulte las instrucciones en el archivo léame de la extensión para obtener información detallada.
